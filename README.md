## GUNDAM FIGHTER 
Player:
     
    Ø Up : Jump
    Ø Down : Crouch (TBD)
    Ø Right : Move right
    Ø Left: Move left
    Ø P : punch
    Ø Enter : Power move

Opponent:

    Ø W: Jump
    Ø S: Crouch (TBD)
    Ø D : Move right
    Ø A: Move left
    Ø Q : Attack
